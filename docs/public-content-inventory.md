# Public Content Inventory

## Editable in CMS v1

- `site.header`
  - brand name and tagline
  - nav labels
  - auth CTA labels
- `site.footerBrand`
  - footer title, tagline, brand description
- `site.footerExplore`
  - block title and explore links
- `site.footerCompany`
  - block title and company links
- `site.contactInfo`
  - address, phone number, email, business hours
- `site.socialLinks`
  - facebook, instagram, youtube URLs
- `site.seo`
  - default meta description

- `home.hero`
  - badge, headline, description, CTA text
- `home.stats`
  - four marketing stats
- `home.featuredToursIntro`
  - badge, section title, intro copy, view-all CTA
- `home.commitments`
  - title, description, four commitment items
- `home.finalCta`
  - title, description, CTA texts

- `about.hero`
  - badge, title, description
- `about.story`
  - story title and two paragraphs
- `about.missionVision`
  - title, mission, vision, core values
- `about.team`
  - title, description, four member name/role pairs

- `contact.hero`
  - badge, title, description
- `contact.cards`
  - titles for address, phone, email, working hours cards
- `contact.formIntro`
  - form title, description, submit label

- `publicTours.indexHero`
  - hero title, description, search placeholder
- `publicTours.emptyState`
  - title, description, CTA text

- `booking.consultationHero`
  - consultation hero title and description
- `booking.consultationBenefits`
  - quick contact title
  - reasons title
  - form title and submit text
  - success title, success description, explore tours CTA
- `booking.statusCopy`
  - success, failed, and error titles/descriptions

## Not managed in CMS v1

- `PublicTours/Details` dynamic tour detail data
- tour cards, prices, ratings, dates, and availability from `Tour`
- booking and payment transactional data from `Booking` and payment flow
- small utility-only labels that do not affect business content

## Admin surfaces

- `Admin/Content/Index`
  - site-wide settings tab
  - page tabs: Home, About, Contact, Tours, Booking
- right sidebar message center in admin layout
